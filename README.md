# 自用的X86和RockChip固件


## 说明

- 编译固件结束后，自动发送WeTransfer下载地址到Bark.
- 自用固件比较大，因此在menuconfig中选择了“gzip”进行压缩，压缩后150兆左右，解压后大小3G左右。注意需要用WinRAR解压，用7zip会解压失败。
- 感谢Lean提供的开源代码和p3terx大神的Action教程。
